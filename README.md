## Installation

1. Install dependencies:
```bash
npm install
```

## Usage

Run the application:
```bash
npm start
```

Run in development mode (with DevTools):
```bash
npm run dev
```

## Customization

- **Window Size**: Edit `width` and `height` in `main.js`
- **Transparency Level**: Adjust `rgba` values in `styles.css`
- **Colors**: Modify gradient colors in the CSS
- **Content**: Edit `index.html` to add your own content
